# 🌉 MemoBridge Skill

AI 记忆迁移技能 | AI Memory Migration Skill

## 功能 / Features

在 AI 工具之间自由迁移记忆：CodeBuddy / OpenClaw / Hermes / Claude Code / Cursor / ChatGPT / 豆包 / Kimi

## 安装 / Install

将 `memo-bridge/` 目录复制到 `.codebuddy/skills/` 下即可。

```bash
cp -r skills/memo-bridge ~/.codebuddy/skills/
```

## 使用 / Usage

在 CodeBuddy 中说：
- "帮我导出 AI 记忆"
- "把记忆迁移到 Claude Code"
- "memory export"

## 依赖 / Dependencies

- [memo-bridge](https://github.com/gonelake/memo-bridge) CLI (Node.js >= 22)
